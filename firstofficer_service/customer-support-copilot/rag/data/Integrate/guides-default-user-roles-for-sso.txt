# Set default user roles for SSO

You will need to be an admin user andconfigure SSOwith a provider first.
Admins can set default roles for new users joining
the Atlan workspace via SSO. Setting the default role to admin, member, or guest will provide the appropriate permissions to users as soon as they log into Atlan.
To set a default user role for SSO:
- In the left menu from any screen in Atlan, clickAdmin.
- UnderAdmin center, clickSSO.Ã‚
- On the SSO page for your provider, underDefault Role, click the dropdown menu.
- From the dropdown menu, selectGuest,Member, orAdminas the default role for your users.
